============
 Gevangenpoort base
============
------------------------------
 An installable theme for Plone 5.2.x with Python 3
------------------------------

`Latest version on PyPI <https://pypi.python.org/pypi/plonetheme.gevangenpoort>`_

Installation
============================
Install plonetheme.gevangenpoort by adding it to your buildout::

    [buildout]

    ...

    eggs =
        plonetheme.gevangenpoort

and then running ``bin/buildout``
